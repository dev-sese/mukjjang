## 🛠️ 기술 스택

### Frontend
- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion (애니메이션)

### Backend
- Next.js API Routes
- Prisma ORM
- NextAuth.js (인증)

### Database
- PostgreSQL (또는 MongoDB)

### 배포 & 인프라
- Vercel (프론트엔드)
- Railway/Supabase (데이터베이스)

### 분석 & 모니터링
- Google Analytics
- 자체 이벤트 트래킹
